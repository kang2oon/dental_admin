HTML Intro

웹표준 개발의 출발점인 HTML(Hyper Text Markup Language)에 대한 사용 가이드입니다.
  • Markup 개요 : 웹표준 개발의 필요성과 기본적인 용어에 대한 안내로 구성
  • DTD 및 인코딩 : DTD에 대한 정의 및 기본 형식, 인코딩 선언에 대한 정보로 구성
  • 퍼블리싱 규칙 : 웅진 그룹 관련 사이트 개발 진행시 공통으로 사용할 규칙에 대한 정의
  • 코드 작성 규칙 : 코드 퍼블싱에 대한 기본 규칙 및 각 엘리먼트별 작성 규칙에 대한 정의
