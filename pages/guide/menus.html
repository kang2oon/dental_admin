메뉴 구성법

좌측 사이드바 메뉴의 구성은 XML 데이터를 호출하는 방식으로 제작하여 관련 파일만 수정하여 전체 프레임워크에서 제어할 수 있도록 구성.
관련 파일은 /menus/ 폴더 하위 파일 
[
{
  "text":"HTML",
  "state":"open",
  "children":[{
	"text":"HTML Intro"
	},{
	"text":"Markup 개요"
	},{
	"text":"DTD 및 인코딩"
	},{
	"text":"퍼블리싱 규칙"
	},{
	"text":"코드 작성 규칙"
  }]
},
{
  "text":"CSS",
  "state":"closed",
  "children":[{
	"text":"CSS Intro"
	},{
	"text":"CSS 코드 작성 규칙"
	},{
	"text":"CSS Selector"
	},{
	"text":"RESET CSS"
	},{
	"text":"CSS 폰트 단위"
	},{
	"text":"CSS Hack"
  }]
}
]
상기 샘플과 같이 XML 구조화된 메뉴 구조를 이용하여 죄측 사이드 영역에 트리형 메뉴 구조 생성
  • "text":에 메뉴명 및 링크 주소 연결 코드 삽입
  • 하부 메뉴가 있는 경우 "children":[{을(를) 이용한 계층화 구조 생성
  • "state":를(을) 이용하여 계층 구조의 open, closed 제어
  • "page_open('/markup/hack', this) 부분의 앞쪽 변수에 /pages/ 폴더 하단의 연결하고자 하는 파일 작성
    (단, 확장자 제외)
