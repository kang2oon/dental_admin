콘텐츠 생성

콘텐츠 페이지의 파일명은 메뉴 구성법에서 페이지의 "page_open('/markup/hack', this)에 기입된 폴더와 파일명으로 동일하게 생성한다.
확장자는 .html 로 제작한다.

제작 방법

  1. 기본 템플릿 다운로드
    샘플파일 다운로드
  2. 파일명 수정 : 죄측 메뉴에 명명한 변수와 동일한 파일명으로 수정
  3. 타이틀 수정 및 콘텐츠 내용 수정
  4. 변경 이력 등록
<div id="content">
  <h2>타이틀</h2>
  <div class="mgtop_low">
	내용
  </div>
  <hr/>
  <h4 class="history">변경이력</h4>
  <ol class="type_num li_inside">
	<li>최초 제정 : 2015-01-12 강지영</li>
  </ol>
</div>

제작 Tip

  • 기본 스타일 시트에서 자주 사용되는 클래스 숙지
  • Syntax Highlight 사용시 "<"는 코드값( & l t ; )을 이용하여 작성

변경이력

  1. 콘텐츠 업데이트 : 2015-01-13 강지영
  2. 최초 제정 : 2015-01-12 강지영
